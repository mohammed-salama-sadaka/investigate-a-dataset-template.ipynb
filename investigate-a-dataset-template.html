Introduction¶

Research Question 2 (How to determine if a player is a defender or attacker based on player attributes?)¶

Determine if a player is a defender or an attacker. I will drop players that arn't neither defenders nor attackers.

In [15]:
def classify_player( row ):
    if row['defensive_work_rate'] == 'high':
        return 'defender'
    elif row['attacking_work_rate'] == 'high':
        return 'attacker'
    else:
        return 'no'
dataframe['category'] = dataframe.apply( classify_player, axis= 1 )
In [16]:
dataframe = dataframe.drop(dataframe[dataframe['category'] == 'no'].index)

Drop player_api_id column from my dataframe because it's not needed for answering the second question.

In [17]:
dataframe.drop(columns= dataframe[['player_api_id','defensive_work_rate','attacking_work_rate','weight']],inplace= True)

Conclusions¶

  1. As body weight increases for a player the balance and sprint speed decreases in general.
  2. Body weight doesn't have a significant effect on stamina for a player.
  3. On average, attacker players are higher in balance, sprint speed, dribbling, ball control shot power and vision.
  4. On average, defender players are higher in stamina, strength, standing tackle and long pass.
